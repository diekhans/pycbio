* generate gnome teat regions:
** create a BED4 for the region
** extract sequence with
twoBitToFa -bedPos -bed=
** rename name regions to be in the form:
chr19_51887540_51906200
** BLAT to get mappings of region to genome
blat -q=dna -t=dna -noHead the.2bit:chr1 region.fa reg1.psl
** pick best alignments
** project annotations with either
- pslMap: use -swapMap to project to region
- liftOver: swapPsl and pslToChain
- CrossMap for BAMs
